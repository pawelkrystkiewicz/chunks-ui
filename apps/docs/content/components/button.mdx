# Button

A styled button component built on Base UI's `Button` primitive. Supports multiple visual variants, semantic colors, loading state, and start/end icons.

## Import

```tsx
import { Button } from 'chunks-ui'
```

## Basic Usage

```tsx
<Button>Click me</Button>
```

<Container>
  <Button>Click me</Button>
</Container>

## Variants

The `variant` prop controls the visual style:

```tsx
<Button variant="contained">Contained</Button>
<Button variant="outlined">Outlined</Button>
<Button variant="text">Text</Button>
```

<Container>
  <Button variant="contained">Contained</Button>
  <Button variant="outlined">Outlined</Button>
  <Button variant="text">Text</Button>
</Container>

## Colors

The `color` prop sets the semantic color:

```tsx
<Button color="primary">Primary</Button>
<Button color="destructive">Destructive</Button>
<Button color="success">Success</Button>
<Button color="warning">Warning</Button>
<Button color="secondary">Secondary</Button>
```

<Container>
  <Button color="primary">Primary</Button>
  <Button color="destructive">Destructive</Button>
  <Button color="success">Success</Button>
  <Button color="warning">Warning</Button>
  <Button color="secondary">Secondary</Button>
</Container>

Colors work with all variants:

```tsx
<Button variant="outlined" color="destructive">Delete</Button>
<Button variant="text" color="success">Approve</Button>
```

<Container>
  <Button variant="outlined" color="destructive">
    Delete
  </Button>
  <Button variant="text" color="success">
    Approve
  </Button>
</Container>

## Loading State

When `loading` is `true`, the button shows a `Loader` spinner in place of the `startIcon` and becomes disabled:

```tsx
<Button loading>Saving...</Button>
```

<Container>
  <Button loading>Saving...</Button>
</Container>
The button remains focusable while loading (for accessibility) but click events are suppressed.

## Icons

```tsx
<Button startIcon={<PlusIcon />}>Add Item</Button>
<Button endIcon={<ArrowRightIcon />}>Next</Button>
```

<Container>
  <Button startIcon={<PlusIcon />}>Add Item</Button>
  <Button endIcon={<ArrowRightIcon />}>Next</Button>
</Container>
Icons are hidden during loading state. The `startIcon` slot is replaced by the `Loader`.
