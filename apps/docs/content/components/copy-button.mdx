# CopyButton

A button that copies a value to the clipboard and shows a brief confirmation. Transitions from a clipboard icon to a check icon on click.

## Import

```tsx
import { CopyButton } from "chunks-ui";
```

## Basic Usage

```tsx
<CopyButton value="npm install chunks-ui" />
```

<CopyButtonBasicExample />

## Custom Children

Use the render-prop `children` for custom content. The `copied` boolean reflects the current state:

```tsx
<CopyButton value="REF-2024-XK9">
  {({ copied }) => (
    <span className="text-xs">{copied ? "Copied!" : "Copy code"}</span>
  )}
</CopyButton>
```

<CopyButtonCustomExample />

## Timeout

Control how long the "copied" confirmation persists with the `timeout` prop (default 2000ms):

```tsx
<CopyButton value="text" timeout={5000} />
```
