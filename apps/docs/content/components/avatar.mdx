# Avatar

Displays a user avatar image with automatic fallback to initials. Supports numeric sizing and multiple shapes.

## Import

```tsx
import { Avatar } from 'chunks-ui'
```

## Basic Usage

With an image:

```tsx
<Avatar src="/jane.jpeg" alt="Jane Doe" />
```

Without an image (shows initials derived from `alt`):

```tsx
<Avatar alt="Jane Doe" />
```

<Container>
  <Avatar src={'/jane.jpeg'} alt="Jane Doe" />
  <Avatar alt="Jane Doe" />
</Container>

This renders "JD" as fallback initials.

## Custom Fallback

Override the auto-generated initials with the `fallback` prop:

```tsx
<Avatar fallback="?" />
<Avatar alt="John Doe" fallback="J" />
<Avatar alt="John Doe" fallback="J" src={'/bro2.jpeg'}/>
```

<Container>
  <Avatar fallback="?" />
  <Avatar alt="Jane Doe" fallback="J" />
  <Avatar alt="John Doe" fallback="J" src={'/bro2.jpeg'} />
</Container>

## Sizes

The `size` prop accepts a numeric value in pixels:

<div className="flex items-center justify-between max-w-lg mx-auto">
  {[10, 20, 30, 40, 50, 60, 100].map(size => (
    <div className="grid grid-rows-2 gap-x-10 gap-y-5 items-center h-[200px]" key={size}>
      <p className="mx-auto">{size}px</p>
      <Avatar src={'/jane.jpeg'} size={size} className="self-center" />
    </div>
  ))}
</div>

## Shapes

```tsx
<Avatar alt="User" shape="circle" size={50}/>
<Avatar alt="User" shape="rounded" size={50}/>
<Avatar alt="User" shape="square" size={50}/>
```

<Container>
  <Avatar alt="User" shape="circle" src={'/bro.jpeg'} size={50} />
  <Avatar alt="User" shape="rounded" src={'/bro.jpeg'} size={50} />
  <Avatar alt="User" shape="square" src={'/bro.jpeg'} size={50} />
</Container>

## Initials Generation

When no `src` is provided and no `fallback` is set, initials are automatically generated from the `alt` text:

- `"Jane Doe"` produces `"JD"`
- `"Alice B. Charlie"` produces `"AB"` (first two initials)
- Single word `"Admin"` produces `"A"`
