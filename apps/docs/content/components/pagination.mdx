# Pagination

A navigation component for paginated content. Uses semantic HTML (`<nav>`, `<ul>`, `<li>`) with `aria-label="pagination"`. Unstyled structure — consumers provide page range logic.

## Import

```tsx
import { Pagination } from "chunks-ui";
```

## Basic Usage

```tsx
<Pagination.Root>
  <Pagination.Content>
    <Pagination.Item>
      <Pagination.Previous href="?page=1" />
    </Pagination.Item>
    <Pagination.Item>
      <Pagination.Link href="?page=1">1</Pagination.Link>
    </Pagination.Item>
    <Pagination.Item>
      <Pagination.Link href="?page=2" active>2</Pagination.Link>
    </Pagination.Item>
    <Pagination.Item>
      <Pagination.Link href="?page=3">3</Pagination.Link>
    </Pagination.Item>
    <Pagination.Item>
      <Pagination.Next href="?page=3" />
    </Pagination.Item>
  </Pagination.Content>
</Pagination.Root>
```

## With Ellipsis

Use `Pagination.Ellipsis` for large page ranges:

```tsx
<Pagination.Item>
  <Pagination.Ellipsis />
</Pagination.Item>
```

## Custom Link Component

Use the `render` prop on `Link`, `Previous`, and `Next` to render framework-specific links:

```tsx
<Pagination.Link render={<Link to="?page=2" />} active>2</Pagination.Link>
<Pagination.Previous render={<Link to="?page=1" />} />
```

## Compound Parts

| Part | Description |
|---|---|
| `Pagination.Root` | `<nav>` wrapper with `aria-label="pagination"`. |
| `Pagination.Content` | `<ul>` containing page items. |
| `Pagination.Item` | `<li>` wrapping a link, ellipsis, or nav button. |
| `Pagination.Link` | Page number link. `active` sets `aria-current="page"`. |
| `Pagination.Previous` | Previous page button with chevron icon. |
| `Pagination.Next` | Next page button with chevron icon. |
| `Pagination.Ellipsis` | Collapsed page range indicator. |
