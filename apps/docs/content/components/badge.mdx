import { Container } from '@/components'

# Badge

A small informational label for status indicators, categories, and tags. Shares the same color token convention as `Chip` but is non-interactive.

## Import

```tsx
import { Badge } from "chunks-ui";
```

## Basic Usage

```tsx
<Badge>Default</Badge>
```

<Container>
  <Badge>Default</Badge>
</Container>

## Colors

```tsx
<Badge color="primary">Primary</Badge>
<Badge color="destructive">Destructive</Badge>
<Badge color="success">Success</Badge>
<Badge color="warning">Warning</Badge>
<Badge color="secondary">Secondary</Badge>
```

<Container>
  <Badge color="primary">Primary</Badge>
  <Badge color="destructive">Destructive</Badge>
  <Badge color="success">Success</Badge>
  <Badge color="warning">Warning</Badge>

</Container>

## Outlined Variant

```tsx
<Badge variant="outlined" color="primary">Outlined</Badge>
<Badge variant="outlined" color="success">Available</Badge>
```

<Container>
  <Badge variant="outlined" color="primary">Outlined</Badge>
  <Badge variant="outlined" color="success">Available</Badge>
  <Badge variant="outlined" color="destructive">Discontinued</Badge>
</Container>

## Props

| Prop | Type | Default | Description |
|---|---|---|---|
| `color` | `"primary" \| "destructive" \| "success" \| "warning" \| "secondary"` | `"secondary"` | Semantic color variant |
| `variant` | `"contained" \| "outlined"` | `"contained"` | Visual style |
| `className` | `string` | -- | Additional CSS classes |

All other props are forwarded to the root `<span>` element.
